# ThaiDeals Affiliate Shop (PHP Version)

สคริปต์รวมสินค้าดีลพิเศษ เวอร์ชัน PHP ธรรมดาที่สามารถรันได้บนโฮสติ้งทั่วไป (Shared Hosting) ทุกประเภท เช่น cPanel, DirectAdmin โดยไม่ต้องใช้ Node.js

## คุณสมบัติ
- **รันง่าย:** รองรับ PHP 7.4 ขึ้นไป ไม่ต้องติดตั้ง dependencies เพิ่มเติม
- **จัดการง่าย:** มีหน้า Admin สำหรับตั้งค่าชื่อเว็บ หมวดหมู่ และอัปโหลดไฟล์ CSV
- **รวดเร็ว:** ใช้ระบบไฟล์ (JSON/CSV) แทนฐานข้อมูล ทำให้โหลดข้อมูลได้ไวและย้ายโฮสต์ได้ง่าย
- **Responsive:** รองรับการแสดงผลทุกหน้าจอด้วย Tailwind CSS

## วิธีการติดตั้ง
1. อัปโหลดไฟล์ทั้งหมดขึ้นไปยังโฮสติ้งของคุณ (ในโฟลเดอร์ `public_html` หรือโฟลเดอร์ที่ต้องการ)
2. ตรวจสอบสิทธิ์ (Permission) ของโฟลเดอร์ `data/` ให้สามารถเขียนไฟล์ได้ (แนะนำเป็น 755 หรือ 777)
3. เข้าใช้งานผ่านเบราว์เซอร์ได้ทันที

## การจัดการข้อมูล
- **หน้า Admin:** เข้าไปที่ `yourdomain.com/admin.php` เพื่อตั้งค่าเว็บไซต์
- **ไฟล์ CSV:** 
  - อัปโหลด `main.csv` สำหรับสินค้าหน้าแรก
  - อัปโหลดไฟล์ CSV แยกตามหมวดหมู่ได้ในหน้า Admin
- **โครงสร้าง CSV:** ต้องมีหัวตาราง (Header) ดังนี้:
  `product_id,product_name,product_price,product_discounted,product_discounted_percentage,product_image,tracking_link,category_name`

## ความปลอดภัย
- มีไฟล์ `.htaccess` ป้องกันการเข้าถึงโฟลเดอร์ `data/` โดยตรง
- แนะนำให้เปลี่ยนชื่อไฟล์ `admin.php` เป็นชื่ออื่นเพื่อความปลอดภัยเพิ่มเติม
